# Mouse Tracker Extension - Plasmo

這是一個使用 [Plasmo Framework](https://docs.plasmo.com/) 所開發的瀏覽器擴充功能，用於追蹤使用者滑鼠在網頁上的行為。  
擴充功能能即時記錄滑鼠移動、停留位置與持續時間，並支援匯出為 JSON 檔案，適合用於使用者行為分析、UX 優化或互動熱區研究。

---

## 📌 功能特色

- 🖱 **滑鼠移動追蹤**  
  實時監測滑鼠在畫面中的移動軌跡與座標紀錄

- ⏱ **停留時間偵測**  
  當滑鼠在同一點停留超過 **2 秒**，自動記錄該位置與時間戳

- 💾 **本地儲存**  
  所有紀錄自動儲存至 `localStorage`，關閉頁面後仍保留資料

- 📤 **一鍵匯出 JSON 檔案**  
  透過 popup UI 提供匯出按鈕，下載所有追蹤資料（含時間戳與座標）

- 🧩 **Plasmo 開發架構**  
  使用 Plasmo 快速構建 extension，支援熱重載與 TypeScript 開發體驗

---

## 🧱 技術架構

- **前端框架**：Plasmo + React + TypeScript
- **滑鼠事件監聽**：`mousemove`、`mouseenter`、`mouseleave`
- **資料儲存**：`localStorage`
- **匯出功能**：Blob + anchor `download` 實現 JSON 匯出

---

## 🖼 使用情境

- 🔥 **網頁熱區視覺化（可搭配 heatmap 分析）**
