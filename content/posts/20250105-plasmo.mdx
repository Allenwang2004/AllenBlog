# Mouse Tracker Extension - Plasmo

這是一個使用 [Plasmo Framework](https://docs.plasmo.com/) 所製作的瀏覽器擴充功能，能夠 **追蹤使用者滑鼠在網頁上的停留位置與時間**，並可透過彈出視窗匯出追蹤資料。

---

## 功能特色

- 追蹤滑鼠移動與停留座標
- 偵測滑鼠在某點停留超過 2 秒即記錄
- 自動儲存追蹤資料至 local storage
- 支援一鍵匯出成 JSON 檔案
