**AI 助理開發平台 [Github demo](<https://github.com/Allenwang2004/Allagent>)**

---

### 一、主要目標

打造一個基於 Web 的 AI 助理開發平台，可執行 RAG 查詢、執行 LLM 問答，擁有簡易式的 Web Chat UI 和短接線的知識庫管理效能。

---

### 二、基本功能模組 (Phase 1)

1. **UI 前端**
   - React (Next.js) + TailwindCSS
   - 使用 Flask 連線
   - 支援問答
   - 文件上傳

2. **背後 API**
   - Flask
   - Endpoint:
     - `/chat`: 接受問題、經 RAG 處理后返回 LLM 答案
     - `/upload`: 上傳文件 (對應知識庫)

3. **RAG + LLM 問答**
   - `LangChain` 處理:
     - DocumentLoader -> TextSplitter -> Embedding (OpenAI / HuggingFace)
   - 向量庫: FAISS
   - LLM: Ollma

4. **後台**
   - 存放已上傳的資料

---

### 三、未來擴充 (推薦 Phase 2+)

- 支援官方 LINE bot / Messenger bot
- 文本通訊分析 (大量實例 log analysis)
- TTS / STT (聲音對話)
- Plugin 式 Function Calling / API Agent
- 多人語境、不同角色助理分辨
